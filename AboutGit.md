## Create Repository

| $ mkdir name             | 创建 repository                 |
| :----------------------- | :------------------------------ |
| **$ cd learngit**        | **进入 repository**             |
| **$ pwd**                | **显示当前目录**                |
| **$ git init**           | **转为git可以管理的repository** |
| **$ git add readme.txt** | **将文件添加到repository**      |
| **$ git commit -m**      | **-m后输入本次提交的说明**      |
| q                        | 退出                            |
